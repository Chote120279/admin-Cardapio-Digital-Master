# admin-Cardapio-Digital-Master

Aplicação de administração para cardápio digital.

## Instalação

```bash
npm install
```

## Desenvolvimento

Execute o servidor de desenvolvimento:

```bash
npm start
```

Navegue para `http://localhost:4200/`. A aplicação será recarregada automaticamente se você alterar qualquer arquivo fonte.

## Build

Execute para fazer o build do projeto:

```bash
npm run build
```

Os artefatos de build serão armazenados no diretório `dist/`.

## Testes

Para executar os testes:

```bash
npm test
```

Para mais informações sobre como testar, consulte o [Guia de Testes](TESTING.md).
